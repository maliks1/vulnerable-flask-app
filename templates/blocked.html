ℹ️ Cara Kerja Middleware
  1. Request POST /protected-login diterima Flask.
  2. Hook @before_request mengambil nilai setiap field form dan mengirimkannya ke model Naive Bayes yang sudah dilatih.
  3. Model menghasilkan probabilitas per kelas menggunakan predict_proba(). Probabilitas kelas SQLi = {{ (confidence * 100) | round(2) }}%.
  4. Karena probabilitas ≥ threshold blokir (50%), request dihentikan sebelum query SQL dikonstruksi — database tidak pernah menerima input berbahaya ini.
